Working with Questions that have Biblical Terms

One of the ways that Transcelerator helps with the translation of questions is by automatically supplying translations for words in questions that it identifies as biblical terms (sometimes also referred to as “key terms”). If a thorough analysis of the Major Biblical Terms list has been completed in Paratext, all the identified renderings for each term will be available to Transcelerator. For any terms that do not yet have identified renderings Paratext attempts to do a statistical analysis to find likely renderings, so in some cases there can be omissions or incorrect renderings included.

When you select a question in the main window in Transcelerator, if it contains biblical terms, the Biblical Terms Pane will display a list of potential vernacular renderings for each occurrence of each term. If there are no known renderings, the list will be empty, but the English term will still be shown. If the same term occurs more than once in the question, the same list of renderings will be displayed for each occurrence, corresponding to their order in the English question. Note: If the source questions are being displayed in a language other than English, it is possible that the order will not correspond to the order of the terms in that language.

For any term that has one or more known renderings, Transcelerator automatically includes the default rendering in its guessed translation. So in the common case of questions that match a “frame” (e.g., What did Jesus say to Peter?) Transcelerator will usually be able to fill in the names to achieve a guessed translation that is correct and merely needs to be confirmed.

To change the rendering used in a particular translation, simply select the desired rendering from the list.

If the needed rendering is not available in the list, do one of the following:

  • Simply type the desired word in the translation, replacing the default rendering. If the rendering you typed is not likely to be useful for other questions that might use that term, there might be very little need to add it as a rendering. Note however that since Transcelerator does not know that the word you typed is a rendering of that term, no rendering will be selected for that occurrence of the biblical term and it will not be possible for Transcelerator to process this translation as model for other questions that might have the same frame.
  • Right-click the corresponding occurrence of the term in the Biblical Terms Pane, and in the context menu, click Add Rendering. In the Add Rendering dialog box, type the desired rendering, and then click OK. The newly added rendering will be selected and will automatically replace the previous rendering in the translation.
  • If the desired rendering is one that is used in the Scripture translation and you want the Biblical Terms tool in Paratext to include it as a recognized rendering, then do the following:
    1. Right-click the corresponding occurrence of the term in the Biblical Terms Pane, and in the context menu, click Find Term in Paratext.
