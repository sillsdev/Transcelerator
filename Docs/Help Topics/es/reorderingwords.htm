Como reordenar palabras en una traducción

Aunque Transcelerator es capaz de sugerir traducciones para vocablos bíblicos y otras palabras que ha aprendido a medida que se traducen las preguntas, la mayoría de las veces las traducciones que sugiere tendrán las palabras en un orden que refleja el de las palabras en inglés. Para algunos patrones comunes, quizá se podrá añadir reglas de preprocesamiento para adaptar las preguntas para reordenar las partes de las preguntas de forma que coincidan más estrechamente con el orden de palabras en el idioma de destino. Pero aun así encontrará a menudo situaciones en que se necesitan mover ciertas palabras o frases para ponerlas en el orden correcto. Por supuesto, puede hacer esto usando las operaciones de corte y pegar. Sin embargo, Transcelerator proporciona algunos atajos especiales de teclado para ayudarle a reordenar las palabras más rápidamente. Los comandos Mover palabra(s) a la derecha y Mover palabra(s) a la izquierda correspondientes a estos atajos están disponibles en el menú Editar al ingresarse en una celda de Traducción para editarla. Por supuesto, puede usar los comandos del menú, pero éstos existen principalmente para presentar los atajos.

Cuando una sola palabra necesita ser movida a la derecha o a la izquierda para desplazarla a su posición correcta en la traducción, simplemente coloca el punto de inserción en cualquier lugar de la palabra y presiona Alt+Mayús+Flecha derecha o Alt+Mayús+Flecha izquierda. Cuando presione el atajo de teclado por primera vez, la palabra actual será seleccionada — junto con el espacio anterior o posterior — y desplazada una palabra a la derecha o a la izquierda. Cada vez que utilice el atajo, se desplazará una posición más. Transcelerator intenta usar la razón al decidir si incluir o excluir la puntuación adyacente, pero si no está seleccionando el texto correcto automáticamente, puede seleccionar los caracteres exactos que desea mover como se describe a continuación.

Si se necesitan desplazar varias palabras, mover solo una parte de una palabra, o controlar exactamente cuáles espacios o puntuación se deben mover, simplemente seleccione el texto exacto que desea mover antes de presionar Alt+Mayús+Flecha derecha o Alt+Mayús+Flecha izquierda. En este caso, usted tendrá que decidir si incluir los espacios iniciales o finales (u otros caracteres de ruptura de palabra) para obtener el resultado deseado.

Sugerencia: Si utiliza los comandos para reordenar palabras y resulta algo que no deseaba, se puede revertir los cambios y evitar guardar y confirmar la traducción al presionar Esc. Esto descartará todas las ediciones de la traducción, no solo los cambios realizados al reordenar las palabras.
